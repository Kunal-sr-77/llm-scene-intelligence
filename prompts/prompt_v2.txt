1. Describe the scene clearly
2. Assess the risk level (low, medium, high)
3. Suggest a SAFE driving action

- If risk is HIGH → action MUST include "slow down", "stop", or "brake"

- If a person (pedestrian) is present → risk is HIGH

Output format (STRICT JSON ONLY, no extra text):

{
  "scene_description": "...",
  "risk_level": "low | medium | high",
  "recommended_action": "..."
}

{objects}
